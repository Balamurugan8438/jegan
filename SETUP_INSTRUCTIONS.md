# Setup Instructions for Home Background Image

## How to Add the Flower Background Image

Your home page is now ready to display the beautiful flower image!

### Option 1: Add Your Image File (Recommended)
1. **Download or save the flower image** you provided
2. **Save it as `home-background.jpg`** in the `public` folder
3. The path should be: `d:\madhu\public\home-background.jpg`
4. **Refresh your browser** (Ctrl+Shift+Delete for hard refresh)
5. The flower background will now display on the home page!

### Option 2: Convert Image Online
If you need to convert your image format:
1. Visit https://convertio.co/image-converter/ or similar
2. Upload your flower image
3. Convert to JPG format
4. Download and save as `home-background.jpg` in the `public` folder

### Image Size Recommendations
- **Dimensions**: At least 1920x1080 pixels
- **Format**: JPG, PNG, or WebP
- **Size**: Keep under 2MB for faster loading

### What You'll See
- Full-screen flower background image
- Elegant pink overlay gradient
- "Explore More →" button in the center
- Click the button to navigate to other pages

### If Image Doesn't Load
The fallback background color is a beautiful pink gradient (#FFD1E8 to #FFF0F6), so it will still look great!
